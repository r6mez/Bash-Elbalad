- Command List
clear                    - Clears the terminal screen
pwd                      - Prints current working directory
cd     [dir]             - Changes the directory (default to Home/Desktop)
ls     [-l]              - Lists files (long format with -l)
stat   [file]            - Displays file/directory information
touch  [file]            - Creates/updates an empty file
rm     [-f]  [file]      - Removes file/directory (force with -f)
mv     [src] [dst]       - Moves/renames file/directory
cp     [src] [dst]       - Copies file/directory
mkdir  [-p]  [dir]       - Creates directory (-p to create parent dirs)
rmdir  [dir]             - Removes an empty directory
find   [dir] [name]      - Searches files in a directory
exit                     - Exits the shell
help                     - print shell Documentation
cat    [file]            - Displays file contents 
cat    [file] > [file]   - Redirects file contents to another file
